# Brain Tumor Detection from MRI using Computer Vision

## Project Overview
This project implements a **simple computer visionâ€“based deep learning system** to detect brain tumors from MRI scan images. A **Convolutional Neural Network (CNN)** is used to classify MRI images into two categories: **Tumor** and **No Tumor**.

---

## Objectives

- Train a CNN for binary image classification  
- Predict tumor presence from MRI scans  
- Understand medical image analysis using computer vision
